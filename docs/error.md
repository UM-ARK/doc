# Error Code
* 1: request successful
* 10: Partial success
* 2 : data does mot exist
* 20 : required parameter is empty
* 3: Token has expired
* 31:The maximum refreshable token has expired, need to log in again
* 400: Bad Request
* 401: wrong account or password 
* 900 : unknown error
