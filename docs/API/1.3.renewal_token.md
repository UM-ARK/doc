# renewal_token

## Description:
When the token is about to expire, you can use this API to renew it.

## Request URL:
`http://XXX.XXX/api/api/renewal_token/`

method: POST
### POST parameters：
|name     |required|type| illustrate                                        |
|---------|--------|----|---------------------------------------------------|
|token    |yes     |str | Authentication user; the token must set in header |
